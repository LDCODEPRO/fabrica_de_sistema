## Reliability Ratings

- **A — Authoritative:** Standards bodies (NIST, OWASP, ISO), peer-reviewed papers.
- **B — High quality:** Books by recognized domain experts from reputable publishers.
- **C — Practitioner:** Blog posts, conference talks from known security professionals.

All sources in this index are rated A or B.
